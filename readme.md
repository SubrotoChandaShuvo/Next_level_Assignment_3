# Football Ticket Booking System Database

## Project Overview

This project is a simple SQL database for a **Football Ticket Booking System**. It demonstrates basic database design, table creation, data insertion, and SQL queries using PostgreSQL.

## Features

* Manage users
* Store football match information
* Record ticket bookings
* Maintain relationships using primary and foreign keys
* Apply constraints such as `PRIMARY KEY`, `FOREIGN KEY`, `UNIQUE`, and `CHECK`

## Database Tables

* **Users** – Stores user information.
* **Matches** – Stores football match details.
* **Bookings** – Stores ticket booking information.

## Technologies Used

* PostgreSQL
* SQL

## Project Files

* `DDL.sql` – Table creation and constraints.
* `QUERY.sql` – SQL queries for data retrieval.
* `README.md` – Project documentation.

## How to Run

1. Create a PostgreSQL database.
2. Run `DDL.sql` to create the tables.
3. Insert the sample data.
4. Run the queries in `QUERY.sql` to view the results.
